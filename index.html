Technical Summary

  • Input: dry mono audio (+ optional pose/motion or text prompts).
  • Output: 3rd‑order Ambisonics (16 ch, ACN/SN3D) + stereo + binaural.
  • Control: text → spatial parameter mapping (azimuth/elevation/spread/wet gain, etc.).
  • Rendering: binaural with SOFA HRTF; stereo downmix for speakers.
